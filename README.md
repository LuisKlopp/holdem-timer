홀덤 토너먼트 블라인드 타이머 웹앱입니다. 패키지 매니저는 `pnpm` 기준으로 사용합니다.

## Getting Started

의존성 설치:

```bash
pnpm install
```

개발 서버 실행:

```bash
pnpm dev
```

검증 명령어:

```bash
pnpm lint
pnpm build
```

브라우저에서 `http://localhost:3000` 을 열면 됩니다.

## 주요 파일

- `app/page.tsx`
- `components/TimerDisplay.tsx`
- `components/LevelInfo.tsx`
- `components/ControlPanel.tsx`
- `hooks/useBlindTimer.ts`
